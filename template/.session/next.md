# Next Steps

## Getting Started

1. **Tell Claude what you want to build**
   - Example: "I want to build a task management API"
   - Example: "Build me a React dashboard"
   - Example: "Create a CLI tool for..."

2. **Claude will automatically:**
   - Gather requirements (if unclear)
   - Assess risks
   - Create a detailed plan
   - Ask for approval before coding

## Available Commands

| Say This | What Happens |
|----------|--------------|
| "Build [something]" | Starts planning and execution |
| "Continue" | Resumes from last session |
| "Test" | Runs testing agent |
| "Review" | Runs code review |
| "Security" | Runs security audit |
| "Debug [issue]" | Investigates and fixes bugs |
| "Refactor" | Improves code quality |
